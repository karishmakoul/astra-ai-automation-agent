🛡️ Anti-Hallucination Rules enforced by SYSTEM_PROMPT

1
ALWAYS call list_page_methods() for every page/component before writing any test code. Never assume a method exists.
2
ALWAYS call get_fixtures() before deciding which fixtures to use. Never invent fixture names.
3
If a method does not appear in list_page_methods() output — use a different existing method or add a # TODO: add method() comment instead.
4
NEVER use page.locator(), page.goto(), or any raw Playwright API directly in tests. All browser actions go through page object methods.
5
DO NOT assert exact counts or exact text from live data. Always assert existence · ranges · relative comparisons · pattern matches.
